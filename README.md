# Magikk – A Social Media App
### *Feature Introduction & Product Overview*
---
## Overview
Magikk is a modern social media application inspired by platforms like Instagram, focused on delivering a seamless content-sharing experience with real-time interactions and a scalable mobile-first architecture.

- Replication of core social media functionalities in a performant Android app  
- Explore real-time user engagement features  
- Build and learn scalable and modular architecture using modern Android technologies  

---

## Core Features

### User Authentication
- Secure Login & Signup  
- Session management 

---

### Home Feed
- Infinite scrolling post feed  
- Real-time updates  
- Optimized loading with pagination
- Shimmer effect
- Collapsable Bottom navigation bar 

---

### Post Creation
- Upload images with captions  
- Media handling and compression  
- Instant post publishing  

---

### ❤️ Engagement System
- Like / Unlike posts  
- Comment on posts  
- Dynamic engagement updates  

---

### User Profile
- View user posts  
- Profile details and activity  
- Editable user information
- Private / Public Mode
- Search User Profile 

---

### Notifications 
- Activity-based notifications ( Post Like , Follow )  
- Real-time interaction alerts
- FCM Based Notification Handle  

---

## Key Idea
- Smooth and responsive UI  
- Efficient state management using MVVM  
- Optimized network handling for large feeds  
- Scalable architecture for future expansion  

---

## Tech Stack
- **Language:** Kotlin 
- **UI:** XML  
- **Architecture:** MVVM  
- **Networking:** Retrofit 
- **Image Loading:** Glide
- **Database:** Room   

## Backend Stacks 
- Render Server , Supabase ( PostgreSQL DB )

---

## Architecture Overview
- Clean separation of concerns using MVVM  
- ViewModel manages UI state and business logic  
- Repository layer handles API and local data sources
